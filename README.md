## 🚀 About

This repository contains my solutions to various problems from Codeforces.  
The purpose of this repository is to track my progress in competitive programming and improve my problem-solving skills.

Most solutions are written in C++ and cover topics such as greedy algorithms, dynamic programming, graphs, number theory, and data structures.

The solutions reflect my thought process during contests and practice sessions.

## 📁 Repository Structure

The repository is organized to keep solutions clear and easy to navigate.  
Problems are grouped based on contests or problem sets.

Each solution is implemented in C++ and stored as a separate source file.
