## ⚙️ How to Run

Follow these steps to compile and run the C++ solutions.

### Requirements
- A C++ compiler (such as **g++**)
- Support for **C++17**

### Steps

1. **Copy** the source file (for example `file.cpp`).

2. **Compile the program**

```bash
g++ -std=c++17 file.cpp -o solution
```

3. **Run the program**

```bash
./solution
```
